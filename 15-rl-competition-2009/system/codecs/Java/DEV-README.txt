THIS DISTRIBUTION (DEV)
----------------------------
This is the -dev distribution, which means it comes with source code, JavaDoc, and build scripts, etc.  If you are not a developer, you may prefer the non-dev package, as it is less cluttered :)

----------------------------
Quick Install
----------------------------
The quick way to install this codec is to do:
>$ java -jar products/JavaRLGlueCodec.jar --install

Additional information is available at the project's home page:
http://glue.rl-community.org/Home/Extensions/java-codec
